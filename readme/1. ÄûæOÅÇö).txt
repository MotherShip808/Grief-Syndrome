[共通]

1. 「黄昏フロンティア」様のサイトにて9月9日から配布されているグリーフシンドローム Ver.1.10を用意する。

2. Ver.1.10の「griefsyndrome.exe」と同じ場所に、当ツールの「gso_patch.exe」をコピーする。

3. 「gso_patch.exe」を実行する。

　→ 正常に実行されると「griefsyndrome_online.exe」が作成されます。
　→ 以降、gso_patch.exeは不要です。

4. 作成した「griefsyndrome_online.exe」をグリーフシンドロームのフォルダにコピーする。

5. 「gso.dll」をグリーフシンドロームのフォルダにコピーする。


[グリーフシンドロームを一度もクリアしたことがない場合]

6. 同梱のsave0.datをグリーフシンドロームのフォルダの「save」フォルダ内にコピーする。
